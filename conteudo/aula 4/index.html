usando a tag form, criamos um formulario, onde por meio de tags input podemos,
requisitar dados do usuario, a forma de comportamento de cada input depende de seu tipo

input tipo "text" recebem dados em forma de texto, criando assim uma caixa de texto

ao mesmo tempo. ex: < input type="radio" name="nome_do_grupo" value="valor"> Valor (valor é o texto que será apresentado a frente do botão
assim qualquer botão com o nome "nome_do_grupo" estara ligado e apenas 1 deles podera estar ativo.
