CSS e explicações

até o momento estudamos sobre a estruturação de pagina web
contudo AGR veremos sobre as estrutua de estilo e tags CSS

Todo body possui uma estrutura simples onde pode abrigar conteiners(tendo ou não especificação).
os conteiners genericos utilizam da tags
< div>
onde tudo que esta dentro dele pertence a ele e segue suas regras de estilo CSS
< /div>.
toda div pode ser nomeada com um id: assim a tornando
unica para quando se referenciar a apenas ela, sua estrutura fica < div id="nomediv"> < /div>.
toda div pode tambem pode ser nomeado como parte de um grupo/classe: assim qualquer div da mesma classe
sera referenciada ao se chamar o nome daquela classe, sua estrutura fica < div class="nomeclasse"> < /div>
as divs por serem genericas podem ser manipuladas e ficarem em qualquer lugar da pagina a vontade do
desenvolvedor.
tambem é posivel colocar uma div com id que tambem pertença a uma classe. sua estrutura fica:
< div class="nomeclass" id="nomeid"> < /div>


Dentro do body pode-se abrigar tambem os seguintes conteiners especificos:
eles possuem posições proprias ja definidas:

  • < header> cabeçario: contêiner que utilizamos para inserir conteúdo de introdução na página < /header>
  • < nav> define um conjunto de links/botões de navegação < /nav>
  • < section> seção principal de um documento < /section>
  • < article> artigo: Seções que adicionam conteúdos, mas precisam ser relacionados ao principal < /article>
  • < aside> aparte: Seções que adicionam conteúdos, mas precisam ser relacionados ao principal < /aside>
  • < footer> rodapé do site < /footer>
